# Code and Data for *A Stochastic Isotope-distribution Model for Nitrogen-dependent Fine-root Turnover*

This repository contains MATLAB code used to generate figures and tables for the manuscript *A Stochastic Isotope-distribution Model for Nitrogen-dependent Fine-root Turnover*.

## Required data

Some of the figures and tables require the data file:

* `face_dataset.csv`

This file contains root isotope, root-order, treatment, and sampling-date data used in the model fitting and figure generation. The data file is not included in this repository and is currently available upon request.

## Figures

### Figure 1

Figure 1 is a schematic illustration of the stochastic root replacement model and was not generated from MATLAB code.

### Figure 2

Figure 2 is generated by:

* `figure2.m`

This script compares finite stochastic root-replacement simulations with the corresponding continuum isotope-distribution model. It also computes RMSE between the stochastic and continuum distributions across different numbers of simulated roots and isotope-bin widths.

### Figure 3

Figure 3 is generated by:

* `figure3.m`

and requires the data file. This script plots fitted mean isotope trajectories for selected root orders under unfertilized and fertilized treatments. It also plots mean root longevity by root order for the unconstrained exponential, constrained exponential, and hypoexponential lifespan models.

### Figure 4 

Figure 4 is generated by:

* `figure4.m` 

and requires the data file. This script plots root chemistry summaries used to compare treatment- and order-dependent patterns in the data.

## Tables

Tables 1 and 2 are generated by:

* `tables.m` 

and requires the data file. This script collects the fitted parameter values, derived turnover/longevity quantities, and model-comparison summaries reported in the manuscript tables. Although the manuscript presents these results as two separate tables, they are generated together in this repository for convenience.

## MATLAB requirements

The scripts were written and tested in MATLAB. Individual scripts include comments listing required files and any required toolboxes. Some scripts require no additional MATLAB toolboxes beyond base MATLAB.


Random-number seeds are set in stochastic simulation scripts where needed so that the generated figures are reproducible. Minor visual differences may occur across MATLAB versions due to plotting defaults.
